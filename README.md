# FIO Explorer

A complete client-server application built with React and Fastify.

## Technologies Used

### Frontend
- React (Create React App)
- TypeScript
- React Bootstrap
- SCSS for styling
- React Router for navigation

### Backend
- Fastify
- TypeScript
- Swagger with OpenAPI for API documentation
- Environment configuration with dotenv
